# FixBee Landing Page

Static landing page matched to the FixBee Figma UI & Branding file (`node-id=1-2`).

## Run locally

```bash
python3 -m http.server 4173
```

Open `http://localhost:4173`.

## Project files

- `index.html` — page structure matching the Figma sections
- `styles.css` — spacing, colors, and layout from the design
- `script.js` — mobile nav, reveal animations, contact form, year
- `assets/` — logo mark, mascots, icons, phone mockup, team photos

## Notes

- Figma Files / footer Figma icon → the shared branding file
- Prototype → demo video section
- GitHub → Capstone Team 01 org
- Contact form opens the user’s email client (`fixbee.team@gmail.com` in `script.js`)
